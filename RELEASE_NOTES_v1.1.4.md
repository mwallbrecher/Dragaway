# Dragaway v1.1.4

**Your daily tools, one drag away.**

## What's new

- **Drag Apple Mail messages straight from your inbox.** Dragaway catches single or multiple messages, stores them as standard `.eml` files, extracts the readable message locally, and suggests mail-specific actions such as **Summarize**, **Tell Me What to Do**, and **Deadlines & Risks**. The pill appears quickly while the message is prepared in the background.
- **A cleaner, more useful session header.** File type and active AI model now sit in a compact top row, while the file pill uses the full width below. Safari tabs and Mail messages receive contextual icons without changing their actual file format.
- **Choose your exact AI model.** BYOK users can refresh the live model list for OpenAI, Claude, Gemini, Groq, or Ollama and select the precise model used for requests. Dragaway never silently swaps an unavailable selection for another model. The hosted free plan remains a simple fixed **Gemini 2.5** option.
- **Clipboard History gets you back to work faster.** Picking an entry now returns focus to the app you were using. The optional, default-off **Enhanced Access** setting can also paste the chosen text, image, or file immediately; core Dragaway features remain permission-free.
- **Browser image drops now preserve the image.** When a browser exposes both image data and a page URL, Dragaway keeps the thing you visibly dragged instead of saving the page as text.
- **Visual polish.** The menu-bar icon is properly transparent, controls sit tighter in the redesigned header, and layouts remain responsive across UI scales.

## Install / update

Installed copies update in place via **Check for Updates…**, or download **Dragaway-1.1.4.dmg** below. The build is signed with a Developer ID certificate and notarized by Apple, so it opens normally without a Gatekeeper workaround.
