# Dragaway v1.1.2

**Your daily tools, one drag away.**

## What's new

- **Drop Safari (and other non-Chromium) browser tabs into Dragaway.** Dragging a tab from Safari — or any browser that isn't Chromium-based — now lands in the pill just like a file or a link. Chromium browsers (Chrome, Edge, Arc, Brave) already worked; this closes the gap for the rest.

## Install / update

Installed copies update in place via **Check for Updates…**, or download **Dragaway-1.1.2.dmg** below. Signed with a Developer ID certificate and notarized by Apple — it opens normally, no Gatekeeper workaround needed.
