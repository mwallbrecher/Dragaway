# Dragaway v1.1.3

**Your daily tools, one drag away.**

## What's new

- **No permissions needed anymore.** Dragaway no longer asks for Accessibility access — drag detection, hotkeys, and the radial launcher all work out of the box, and it never appears in Privacy & Security. Pressing **Esc** still closes the card whenever it's focused.

## Install / update

Installed copies update in place via **Check for Updates…**, or download **Dragaway-1.1.3.dmg** below. Signed with a Developer ID certificate and notarized by Apple — it opens normally, no Gatekeeper workaround needed.
